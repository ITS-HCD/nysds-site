What is the New York State Design System?

Component Library
A collection of common front-end code grouped together in components that can be shared and reused across multiple codebases, often with design, state, and default behavior built-in.
Design Libraries
A library of icons, styles, components, and variables that can be shared and reused across multiple design files for consistent design and prototyping.
Design Tokens
A collection of design decisions stored in key-value pairs, abstracted so they can easily be used across a wide range of platforms, design and code.
Typography
A collection of standard NYS brand-approved typefaces that need to be used in both design and development product work.
