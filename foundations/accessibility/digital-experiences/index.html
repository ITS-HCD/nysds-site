Resources for making websites and applications accessibile.

  • Creating Accessible Forms – Making forms accessible and understandable for all users
  • Accessibility Insights (Microsoft) – Accessibility testing and reporting suite
  • HTML CodeSniffer – Real-time accessibility checker for code
  • Wave – Web accessibility evaluation tool
  • Google Lighthouse – Web performance and accessibility audit
  • Deque Axe – Chrome extension for accessibility testing
  • A11y CSS – Stylesheet for testing accessibility states
  • NVDA – Screen reader for Windows users
  • Pa11y – Command line accessibility testing tool
