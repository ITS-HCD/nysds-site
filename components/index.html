About

Components

Excelsior offers a library of reusable building blocks for creating consistent, accessible, scalable digital experiences across New York State government sites. They are designed and tested to meet WCAG 2.2 AA compliance so they're accessible. They are styled with design tokens for flexibility. And they are built with web components so they work across platforms and frameworks.

Alert
Alert cards are designed to display critical information, updates, or warnings that require the user's attention.
Avatar
Avatars represent a user visually through an image, initials, or an icon and typically identify user-specific actions.
Checkbox
A checkbox is a form input that allows users to select multiple options independently from a list.
Icon
Visual symbols used to convey meaning or action.
Radiobutton
Form input for selecting one option from a group.
Select
Dropdown menu for selecting a single option.
Textarea
Field for entering multiple lines of text.
Textinput
Field for entering short text strings, like email, number, password, and more.
Toggle
Switch component for enabling or disabling a setting.
