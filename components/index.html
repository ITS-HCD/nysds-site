Web Components

The NYS Design System includes a library of web components that serve as building blocks to help developers create consistent digital products. Each component page includes examples, guidance, accessibility considerations, and options.

Accordion
Vertically stacked list of headers that reveal or hide associated content.
Alert
An alert displays a prominent message, with optional links, at the top of the screen.
Avatar
Avatars visually represent a user or entity as initials, a photo, or an icon.
Back To Top
Provides a button that lets users quickly scroll back to the top of the page.
Badge
Provides a visual indicator of categories.
Button
Used primarily for actions that have an immediate result like "Save", "Close", or "Add".
Checkbox
A checkbox is a form input for users to select options (zero, one, or multiple) from a collection of choices.
Divider
Separates content and visually divides sections within layouts.
File Input
Used for selecting and uploading one or more files from the user’s device.
Global Footer
Provide users with essential information, secondary navigation, legal links, and contact details across all pages.
Global Header
Provide users with consistent access to key features, branding, and primary navigation across all pages.
Icon
Visual symbols are used to concisely convey information and increase visual appearance.
Modal
Displays content in a dialog that appears over the page and requires user interaction before returning to the main content.
Pagination
Enables users to navigate through large sets of content or data divided into smaller, more manageable pages.
Radio Button
Form input for selecting one option from a group.
Select
Dropdown menu for selecting a single option.
Skip Navigation
An accessibility feature that lets keyboard and screen reader users skip navigation and jump to main content.
Stepper
Displays to users how many steps are in a process and indicates their progress.
Text Area
Field for entering multiple lines of text.
Text Input
Field for entering short text strings, like email, number, password, and more.
Toggle
Switch component for enabling or disabling a setting.
Tooltip
Displays a brief message when hover/focus, offering additional information or context.
UNav Footer
Provide users with access to essential NYS links across all pages.
UNav Header
A small NYS-branded header at the top of every page to ensure users they are on a secure NYS site.

Suggest a New Component

Do you have an idea for a new NYS Design System web component? Look through the existing proposals in our GitHub discussions board to see if someone already proposed something similar. If not, feel free to submit one.
