Release Notes

We've made bug fixes since our last release and are happy to share the 1.1.0 release of NYSDS. You can explore and install our published packages here: NYS Design System on NPM.

We did not add any new components in this release, but this we've added native form element functionality to NYSDS form-based web components and have fixed a lot of bugs.
