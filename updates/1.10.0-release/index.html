We shipped two new components: <nys-pagination> and <nys-modal>. Use them to create dynamic experiences for your users.

Eric Steinborn
October 3, 2025
2 min read time

Multiple pages? No problem

The <nys-pagination> component allows you to hook into functionality to present pagination functionality to your apps and sites. For more information, read the full documentation for <nys-pagination> component.

Copy Code
<nys-pagination
  id="pagination"
  name="pagination"
  currentPage=4
  totalPages=10
></nys-pagination>

Emily Gorelik shipped the pagination component!

Modal now available

The <nys-modal> component brings pop up content within a page. Use it for providing context to your existing process flows. For more information, read the full documentation for <nys-modal> component.
