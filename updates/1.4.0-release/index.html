‚òÖ NYSDS v.1.4.0

The New York State Design System turns 1.4.0 today! We are releasing a new component this week: <nys-fileinput>! We optimized the NPM package build which cut the file size in half. The entire @nysds/components package is only 380KB now! We added three new icons, print, progress_activity, and attach_file. We also fixed a few bugs.

Join the conversation on our discussion board: https://github.com/ITS-HCD/nysds/discussions/categories/component-proposals

<nys-fileinput>
