Whatʼs New

Check out the latest updates to the New York State Design System.

  • NYS Design System Release 1.16.0 “Algonquin”

    <nys-video> component released as stable. Added flex-gap classes for better customizing flexbox layouts. Sped up build time, and fixed a boatload of smaller issues.
    by Eric Steinborn
    April 2, 2026

  • NYS Design System Release 1.15.0 “Nye”

    Meet <nys-combobox> and <nys-dropdownmenu>, the two newest components in the NYS Design System! We added 3 new icons to the <nys-icon> component, standardized our JS files, fixed bugs, and added a new language configuration option for <nys-unavheader>.
    by Eric Steinborn
    March 6, 2026

  • NYS Design System Release 1.14.0 “Marcy”

    Happy one year anniversary to the NYS Design System! 🍰 We now lint all our styles before publishing and have provided new methods for customizing the <nys-accordion>. Lots of bug and accessibility fixes in this release too!
    by Eric Steinborn
    February 19, 2026

  • Happy Birthday NYS Design System!

    We're celebrating our one year anniversary! We shipped a new homepage and contribute pages. Contribution can take many forms: New feature requests, bug reports, or hop into our Teams channel.
    by Eric Steinborn
    February 19, 2026

  • NYS Design System Release 1.13.1

    A bug fix and enhancements release. Custom search endpoints in <nys-unavheader>. New @nysds/tokens package available. Strong badges make an appearance.
    by Eric Steinborn
    February 5, 2026

  • NYS Design System Release 1.13.0 “TableTop”

    We're releasing two new components, <nys-datepicker> and <nys-table>. We've made some modifications to the <nys-globalheader> to allow for “user actions.”
    by Eric Steinborn
    January 22, 2026

  • NYS Design System Release 1.12.1

    We are wrapping up 2025 with our 34th release of the year! Design previews of nys-table and nys-datepicker are now widely available in the NYS DS Figma library.
    by Eric Steinborn
    December 31, 2025

  • NYS Design System Release 1.12.0 “Skylight”

    This release completely revamped the tooltip component to be more accessible and easier to use. The old tooltip wrapper has been removed from the package. We also introduced new responsive variants for our popular utility classes.
    by Eric Steinborn
    December 11, 2025

  • NYS Design System Release 1.11.3

    We released our community Figma file! Custom Element Manifest (CEM) now available in the package. CEM allows us to map custom components to VSCode, and eventually automatically create our React wrappers.
    by Eric Steinborn
    November 26, 2025

  • NYS Design System Release 1.11.2

    Primarily a bug fix release. No more console warnings. No more accidental double-clicks. We also cleaned up our Figma files.
    by Eric Steinborn
    November 13, 2025
