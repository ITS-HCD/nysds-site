# Digital Experiences
Resources for making websites and applications accessible.

- <a href="https://webaim.org/techniques/forms/" target="_blank">Creating Accessible Forms – Making forms accessible and understandable for all users</a>
- <a href="https://chromewebstore.google.com/detail/accessibility-insights-fo/pbjjkligggfmakdaogkfomddhfmpjeni" target="_blank">Accessibility Insights (Microsoft) – Accessibility testing and reporting suite</a>
- <a href="https://squizlabs.github.io/HTML_CodeSniffer/" target="_blank">HTML CodeSniffer – Real-time accessibility checker for code</a>
- <a href="https://wave.webaim.org/extension/" target="_blank">Wave – Web accessibility evaluation tool</a>
- <a href="https://developers.google.com/web/tools/lighthouse" target="_blank">Google Lighthouse – Web performance and accessibility audit</a>
- <a href="https://chromewebstore.google.com/detail/axe-devtools-web-accessib/lhdoppojpmngadmnindnejefpokejbdd" target="_blank">Deque Axe – Chrome extension for accessibility testing</a>
- <a href="https://ffoodd.github.io/a11y.css/" target="_blank">A11y CSS – Stylesheet for testing accessibility states</a>
- <a href="https://www.nvaccess.org" target="_blank">NVDA – Screen reader for Windows users</a>
- <a href="https://pa11y.org" target="_blank">Pa11y – Command line accessibility testing tool</a>
- <a href="https://www.npmjs.com/package/accessibility-checker" target="_blank">Accessibility-checker – NodeJS module for accessibility scanning</a>
