NYS Design System Release 1.18.3

Fixed a ton of bugs, made the DS more accessible, and improved the MCP Server

Eric Steinborn
May 28, 2026
1 min read time

New <nys-icon>: "delete"

Copy Code
<nys-icon
  name="delete"
  size="5xl"
></nys-icon>

Bug fixes

  • Consolidate MCP token tools from 4 to 3
  • Remove performance-heavy resize listeners
  • Spacebar styling fixes for <nys-button>
  • Fix an a11y issue with <nys-avatar>
  • <nys-backtotop> now targets the body tag
  • Ghost and text button icon spacing fixes
  • Enhancement to <nys-stepper> documentation
  • Added instructions for getting latest MCP updates
  • Branded custom test reporter implemented
  • New listener for out-of-bounds clicks to close global header menu
  • Remove double focus a11y VO on <nys-checkbox>
  • Fix a11y keyboard navigation and VO for nys-radiobutton
  • Fix <nys-textinput> "tel" type reset functionality
  • Fix <nys-textinput> "other" not disabling in <nys-checkbox> and <nys-radiobutton>
  • Fix alignment of responsive <nys-breadcrumbs>
  • Fix alignment in <nys-globalheader> gutters and padding to align with Figma
  • Fix grid-gap overlapping and running off screen

Emily Gorelik, Robert Chen, Jesse Gardner, and Eric Steinborn ain't afraid of no bugs!
